# TodoApp Backend (.NET)

This backend contains two .NET projects:

- **TodoApp**: ASP.NET Core Web API for managing todos.
- **TodoApp.Test**: xUnit test project for the API.

## Getting Started

### Prerequisites

- .NET 8.0 SDK or later

### Restore Dependencies

```bash
dotnet restore
```

### Build the Solution

```bash
dotnet build
```

### Run the API

```bash
dotnet run --project TodoApp
```

### Run Tests

```bash
dotnet test
```
